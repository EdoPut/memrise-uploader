How to use Memrise uploader

To use this add-on you just need to go to one of your courses database page and scroll till the bottom until you find a multiple input form with a button labeled "Upload".

Choose your files and click upload, Firefox could stops for few seconds but there's nothing wrong with that.

Firefox will fire a notification when files have been uploaded. Wait a few second and refresh the page to see your database populated.

Only files that have the same name as the word will be uploaded, regardless of the word format. E.g, fruit.mp3, FRUIT.mp3, FrUiT.mp3, fruit.0.mp3, fruit.another.mp3 will match the word fruit and Fruit.

F.A.Q

  • Does this add-on upload all the files I'm throwing at it?

    No, it just search for the words without audio, the next version will let you choose to upload everything instead if you want to upload every file matching a word go to the addon's prefence page.

  • Why audio files doesn't appear immediately in the database page?

    Every time a browser uploads a file to Memrise's server it receives a request to change the page appearence. I simply ignored this request because to me it was useless. Just refresh the page when you see the notification.
