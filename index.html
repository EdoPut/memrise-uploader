Memrise-uploader

A sleek Firefox add-on that let you upload hundreds of files to memrise's server in a click!

View on GitHub Install addon, version 1.2 Install from Mozilla, version 1.1

What is this?

This is an addon to provide Firefox users a multiple upload choice on the [memrise][memrise] site.

How

  1. Create a course with levels
  2. Go to the database page
  3. Scroll to bottom, there are two buttons: a file input and a reassuring "Upload!" button
  4. Choose your files from the file chooser input
  5. Click upload
  6. A notification will pop when the task is finished
  7. Refresh the page
